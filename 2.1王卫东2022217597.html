模型的设计思路：¶

  • 卷积神经网络: CNN 是深度学习中的一种强大的模型，特别适用于图像和视频识别任务。

  • 注意力模块：通过引入注意力机制，网络能够更加关注图像中的重要部分，从而提高特征的表达能力和网络的分类性能。

  • 批量归一化：加速网络训练，提高模型的泛化能力。

  • 最大池化：减少特征维度，降低计算复杂度，同时保留重要特征。

  • 全连接层和 dropout ：将学习到的特征转换为分类结果，dropout 用于减少过拟合。

import torch.nn.functional as F


# 定义注意力模块
class AttentionModule(nn.Module):
    # 初始化函数，设定输入通道数
    def __init__(self, in_channels):
        super(AttentionModule, self).__init__()
        # 1x1卷积用于生成注意力图
        self.conv = nn.Conv2d(in_channels, 1, kernel_size=1)
        # Sigmoid函数将卷积输出转换为注意力权重
        self.sigmoid = nn.Sigmoid()

    # 前向传播函数
    def forward(self, x):
        # 生成注意力权重
        attention = self.sigmoid(self.conv(x))
        # 将注意力权重应用于输入特征图
        return x * attention


# 定义CNN网络
class CNN(nn.Module):
    def __init__(self):
        super(CNN, self).__init__()
        self.conv1 = nn.Conv2d(3, 8, 3, padding=1)  # 第一层卷积，从3通道到8通道
        self.bn1 = nn.BatchNorm2d(8)  # 批量归一化
        self.attention1 = AttentionModule(8)  # 注意力层，用于增强特征的重要部分
        self.pool1 = nn.MaxPool2d(2, 2)  # 最大池化层
        self.conv2 = nn.Conv2d(8, 16, 3, padding=1)  # 第二层卷积，从8通道到16通道
        self.bn2 = nn.BatchNorm2d(16)  # 批量归一化
        self.pool2 = nn.MaxPool2d(2, 2)  # 最大池化层
        self.conv3 = nn.Conv2d(16, 32, 3, padding=1)  # 第三层卷积，从16通道到32通道
        self.bn3 = nn.BatchNorm2d(32)  # 批量归一化
        self.pool3 = nn.MaxPool2d(2, 2)  # 最大池化层
        self.conv4 = nn.Conv2d(32, 64, 3, padding=1)  # 第四层卷积，从32通道到64通道
        self.bn4 = nn.BatchNorm2d(64)  # 批量归一化
        self.pool4 = nn.MaxPool2d(2, 2)  # 最大池化层
        self.fc1 = nn.Linear(64 * 8 * 8, 256)  # 全连接层，将特征图转换为分类结果
        self.dropout1 = nn.Dropout(0.5)  # 防止过拟合的dropout层
        self.fc2 = nn.Linear(256, 4)  # 最终的全连接层，输出4个类别的预测

    # 前向传播函数
    def forward(self, x):
        x = F.relu(self.bn1(self.conv1(x)))  # 应用ReLU激活函数和批量归一化
        x = self.attention1(x)  # 应用注意力机制
        x = self.pool1(x)  # 应用最大池化
        x = self.pool2(
            F.relu(self.bn2(self.conv2(x)))
        )  # 继续卷积、批量归一化和池化操作
        x = self.pool3(
            F.relu(self.bn3(self.conv3(x)))
        )  # 继续卷积、批量归一化和池化操作
        x = self.pool4(
            F.relu(self.bn4(self.conv4(x)))
        )  # 继续卷积、批量归一化和池化操作
        x = x.view(-1, 64 * 8 * 8)  # 调整特征图的形状以适应全连接层
        x = F.relu(self.dropout1(self.fc1(x)))  # 应用ReLU激活函数和dropout
        x = self.fc2(x)  # 输出最终的分类结果
        return x

from sklearn.metrics import f1_score, accuracy_score
import copy

train_losses = []  # 用于储存训练集的 loss
valid_losses = []  # 用于储存验证集的 loss
accuracies = []  # 用于储存 accuracy
f1_scores = []  # 用于储存 f1 score
best_val_loss = float("inf")
# best_model_wts = copy.deepcopy(model.state_dict())

# 训练模型
for epoch in range(num_epochs):

    start_time = time.time()  # 记录开始训练时间

    # 训练阶段
    model.train()
    train_loss = 0
    for inputs, labels in train_dataloader:
        inputs = inputs.to(device)
        labels = labels.to(device)
        optimizer.zero_grad()
        logits = model(inputs)
        loss = loss_fn(logits, labels)
        loss.backward()
        optimizer.step()
        train_loss += loss.item()

    # 验证阶段
    model.eval()
    valid_loss = 0
    all_preds = []
    all_labels = []
    with torch.no_grad():
        for inputs, labels in valid_dataloader:
            inputs = inputs.to(device)
            labels = labels.to(device)
            logits = model(inputs)
            loss = loss_fn(logits, labels)
            valid_loss += loss.item()

            preds = torch.argmax(logits, dim=1)
            all_preds.extend(preds.cpu().numpy())
            all_labels.extend(labels.cpu().numpy())

    train_losses.append(train_loss / len(train_dataloader))
    valid_losses.append(valid_loss / len(valid_dataloader))
    acc = accuracy_score(all_labels, all_preds)
    accuracies.append(acc)
    f1 = f1_score(all_labels, all_preds, average="weighted")
    f1_scores.append(f1)

    end_time = time.time()  # 记录结束训练时间

    # 输出指标
    print(f"Epoch {epoch+1}/{num_epochs}", end="  ")
    print(f"Training Loss: {train_losses[-1]:.4f}", end=" , ")
    print(f"Validation Loss: {valid_losses[-1]:.4f}", end=" , ")
    print(f"Accuracy: {acc:.4f}", end=" , ")
    print(f"F1 Score: {f1:.4f}", end=" , ")
    print(f"Used Time: {end_time - start_time:.2f}s")

    # 如果当前 epoch 的 验证loss 更低，那么保存模型
    if valid_losses[-1] < best_val_loss:
        print(
            f"Validation loss decreased ({best_val_loss:.4f} --> {valid_losses[-1]:.4f}). Saving model ...",
            end=" ",
        )
        best_val_loss = valid_losses[-1]
        # best_model_wts = copy.deepcopy(model.state_dict())
        # torch.save(best_model_wts, "../model/wwd_model.pt")
        torch.save(model, "../model/wwd_model.pt")
        print("Model Saved")

from sklearn.metrics import accuracy_score, f1_score, roc_auc_score

# 加载模型
# model.load_state_dict(torch.load("wwd_model.pt"))
model = torch.load("../model/wwd_model.pt")

# 设置模型为 evaluation 模式
model.eval()

true_labels = []  # 用于储存真实的标签
pred_probs = []  # 用于储存预测概率

# 迭代测试数据
with torch.no_grad():
    for inputs, labels in test_dataloader:
        inputs = inputs.to(device)
        labels = labels.to(device)
        logits = model(inputs)
        probs = torch.softmax(logits, dim=1)

        # 将真实标签和预测概率添加到 list 中
        true_labels.extend(labels.tolist())
        pred_probs.extend(probs.tolist())

# 将 list 转换为 numpy array
true_labels = np.array(true_labels)
pred_probs = np.array(pred_probs)


# 计算 Accuracy, F1 score, and AUC score
acc = accuracy_score(true_labels, np.argmax(pred_probs, axis=1))
f1 = f1_score(true_labels, np.argmax(pred_probs, axis=1), average="macro")
auc = roc_auc_score(true_labels, pred_probs, multi_class="ovo")


# 输出结果
print(f"Test Accuracy: {acc:.4f}")
print(f"Test F1 score: {f1:.4f}")
print(f"Test AUC score: {auc:.4f}")
