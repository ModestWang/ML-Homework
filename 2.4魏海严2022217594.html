GPU 内存管理¶

  • torch.cuda.empty_cache(): 清理 GPU 内存缓存，该缓存可能会累积并被不再需要的张量占用。这个命令有助于释放这些未使用的内存，使其可以用于其他张量，尤其是在训练大型模型或处理大数据时非常有益。

清理缓存的重要性¶

在训练过程中定期清理缓存有助于防止 CUDA 内存不足错误，特别是在像 Jupyter 笔记本这样的交互式环境中反复修改和测试模型，或者在脚本中连续运行多个训练循环时。

在本项目中，数据增强是一个关键步骤。通过对图像进行各种随机变换，如水平翻转、垂直翻转、旋转和颜色抖动，我们可以增加训练数据的多样性，提高模型的泛化能力。具体来说，使用的变换包括：

  • transforms.Resize((64, 64))：将所有图像调整为64x64像素，以确保输入数据的一致性。
  • transforms.RandomHorizontalFlip() 和 transforms.RandomVerticalFlip()：随机水平和垂直翻转图像，以增加数据多样性。
  • transforms.RandomRotation(15)：随机旋转图像15度以内，以模拟不同角度下的图像变化。
  • transforms.ColorJitter(brightness=0.2, contrast=0.2, saturation=0.2, hue=0.1)：随机调整图像的亮度、对比度、饱和度和色调，以增强颜色的多样性。
  • transforms.ToTensor()：将PIL图像或numpy数组转换为张量。
  • transforms.Normalize([0.485, 0.456, 0.406], [0.229, 0.224, 0.225])：使用ImageNet数据集的均值和标准差对图像进行标准化，以加快模型收敛并提高稳定性。

    return np.clip(image, 0, 1)

def extract_data(dataloader, num_batches=1):
    images, labels = [], []
    for i, (data, target) in enumerate(dataloader):
        if i >= num_batches:
            break
        images.append(data)
        labels.append(target)
    images = torch.cat(images)
    labels = torch.cat(labels)
    # 将图像展平并转换为NumPy数组
    flattened_images = images.view(images.size(0), -1).numpy()
    labels = labels.numpy()
    return flattened_images, labels

  1. 批量处理数据：
    

      

    • 数据加载器可以按批次加载数据，每次迭代训练时只处理一个批次的数据，从而提高内存利用率和计算效率。

    2. 

  2. 数据打乱（Shuffle）：
    

      

    • 在训练数据加载器中设置shuffle=True，可以在每个epoch开始时随机打乱数据。这样可以防止模型对数据顺序产生依赖，提高模型的泛化能力。
      • 

    • 在测试数据加载器中设置shuffle=False，可以保持数据的顺序一致性，有助于评估模型的性能。
      • 

    

    3. 

  3. 异步数据加载：
    

      

    • 数据加载器可以在后台进程中异步加载数据，避免数据加载成为训练的瓶颈，从而加快训练速度。

class ResidualBlock(nn.Module):
    def __init__(self, in_channels, out_channels, stride=1):
        super(ResidualBlock, self).__init__()
        self.conv1 = nn.Conv2d(in_channels, out_channels, kernel_size=3, stride=stride, padding=1)
        self.bn1 = nn.BatchNorm2d(out_channels)
        self.conv2 = nn.Conv2d(out_channels, out_channels, kernel_size=3, stride=1, padding=1)
        self.bn2 = nn.BatchNorm2d(out_channels)
        
        self.shortcut = nn.Sequential()
        if stride != 1 or in_channels != out_channels:
            self.shortcut = nn.Sequential(
                nn.Conv2d(in_channels, out_channels, kernel_size=1, stride=stride),
                nn.BatchNorm2d(out_channels)
            )
    
    def forward(self, x):
        out = F.relu(self.bn1(self.conv1(x)))
        out = self.bn2(self.conv2(out))
        out += self.shortcut(x)
        out = F.relu(out)
        return out

class ImprovedCNN(nn.Module):
    def __init__(self):
        super(ImprovedCNN, self).__init__()
        self.conv1 = nn.Conv2d(3, 16, 3, padding=1)
        self.bn1 = nn.BatchNorm2d(16)
        self.res1 = ResidualBlock(16, 32)
        self.pool1 = nn.MaxPool2d(2, 2)
        self.res2 = ResidualBlock(32, 64)
        self.pool2 = nn.MaxPool2d(2, 2)
        self.res3 = ResidualBlock(64, 128)
        self.pool3 = nn.MaxPool2d(2, 2)
        self.dropout = nn.Dropout(0.5)
        self.fc1 = nn.Linear(128 * 8 * 8, 512)  # 根据输出大小调整
        self.fc2 = nn.Linear(512, 256)
        self.fc3 = nn.Linear(256, 4)

    def forward(self, x):
        x = F.relu(self.bn1(self.conv1(x)))
      
        x = self.res1(x)
        
        x = self.pool1(x)
        
        x = self.res2(x)
        
        x = self.pool2(x)
       
        x = self.res3(x)
        
        x = self.pool3(x)
        
        x = x.view(-1, 128 * 8 * 8)  # 根据输出大小调整
       
        x = self.dropout(F.relu(self.fc1(x)))
        x = F.relu(self.fc2(x))
        x = self.fc3(x)
        return x

























模型初始化、损失函数、优化器和学习率调度器的详细解释：¶
1. 模型初始化:¶
- 这行代码初始化了 `ImprovedCNN` 模型，并将其移到指定的设备上（如 GPU 或 CPU）。使用 `model.to(device)` 可以确保模型的所有参数都在正确的设备上。
- `ImprovedCNN` 是一个自定义的卷积神经网络模型，包含多个卷积层、残差块和全连接层。


2. 定义损失函数:¶
- `loss_fn = nn.CrossEntropyLoss()` 定义了交叉熵损失函数，这是分类问题中常用的损失函数。
- 交叉熵损失函数在计算预测结果与实际标签之间的差异时，适用于多类分类任务。
- 它期望模型的输出为未归一化的类分数（即 logits），并自动应用 softmax 以计算概率分布。


3. 定义优化器:¶
- `optimizer = optim.Adam(model.parameters(), lr=0.001)` 初始化了 Adam 优化器，并设置了学习率为 0.001。
- Adam（Adaptive Moment Estimation）优化器结合了动量梯度下降和 RMSProp 的优点，在实践中表现非常好。
- `model.parameters()` 返回模型的所有可训练参数，优化器将根据这些参数和梯度更新模型权重。


4. 定义学习率调度器:¶
- `scheduler = optim.lr_scheduler.StepLR(optimizer, step_size=7, gamma=0.1)` 定义了一个学习率调度器，用于动态调整学习率。
- `StepLR` 调度器每经过 `step_size` 个周期（这里是7个周期），将学习率乘以 `gamma`（这里是0.1）。
- 动态调整学习率可以帮助模型在训练过程中更好地收敛，避免陷入局部最优解。


总结:¶
这几行代码共同作用，初始化了模型、定义了损失函数和优化器，并设置了学习率调度器。通过这些步骤，我们为模型的训练过程做好了准备：¶
- 模型将使用 Adam 优化器进行权重更新。¶
- 交叉熵损失函数将衡量模型的预测与真实标签之间的差异。¶
- 学习率调度器将根据预定义的规则动态调整学习率，以促进更有效的训练。¶

logits = model(inputs)：使用模型对输入数据进行前向传播，计算输出logits。logits是模型的未归一化输出（即未经过softmax的类分数）。


计算损失：¶
loss = loss_fn(logits, labels)：使用定义的损失函数计算模型输出与实际标签之间的损失。在这里，使用交叉熵损失函数来衡量模型的预测与真实标签之间的差异。


反向传播和参数更新：¶
optimizer.zero_grad()：在进行反向传播之前，先将优化器的梯度缓存清零。每次调用.backward()计算梯度时，梯度都会累加在现有的梯度上，因此在每次迭代中都需要清零梯度。
loss.backward()：进行反向传播，计算模型参数的梯度。调用loss.backward()会计算损失相对于模型参数的梯度，并存储在模型参数的.grad属性中。
optimizer.step()：使用优化器更新模型参数。调用optimizer.step()会根据当前的梯度和优化算法更新模型的参数，以最小化损失。


记录和打印损失：¶
losses.append(loss.item())：将当前epoch的损失值添加到losses列表中。这里使用loss.item()获取损失值的标量表示，并存储在列表中。
print(f'Loss at epoch {epoch+1}: {loss.item():.4f}')：打印当前epoch的损失值，以监控训练过程中的损失变化。

详细说明¶
训练模型的步骤¶
上述代码定义了一个用于训练模型的函数 train_model。该函数通过迭代训练数据加载器的每个批次来更新模型的权重。以下是对每个步骤的详细解释：


将模型设置为训练模式：¶
model.train() 将模型切换到训练模式。这会启用一些特定于训练的层（如 Dropout 和 BatchNorm）来进行参数更新。


迭代训练数据的每个 epoch：¶
for epoch in range(num_epochs) 开始训练循环，每个 epoch 代表一次完整的训练数据集遍历。


迭代每个批次的数据：¶
for inputs, labels in train_dataloader 在每个 epoch 中，遍历训练数据加载器中的每个批次。train_dataloader 按批次提供数据。


将数据移动到指定设备：¶
inputs = inputs.to(device) 和 labels = labels.to(device) 将输入数据和标签移动到指定的设备（如 GPU），以加速计算。


前向传播计算输出：¶
logits = model(inputs) 通过模型进行前向传播，计算输出 logits。


计算损失：¶
loss = loss_fn(logits, labels) 使用定义的损失函数 loss_fn 计算模型输出与真实标签之间的损失。这里使用的是交叉熵损失函数。


梯度清零：¶
optimizer.zero_grad() 清除优化器中所有参数的梯度缓存。每次迭代时，梯度会累积，因此需要在每次反向传播之前清零。


反向传播计算梯度：¶
loss.backward() 执行反向传播，计算损失相对于模型参数的梯度，并存储在每个参数的 .grad 属性中。


更新模型参数：¶
optimizer.step() 使用优化器更新模型的参数。优化器根据计算出的梯度和学习率等超参数，调整模型参数以最小化损失。


打印当前 epoch 的损失值：¶
通过这些步骤，模型的参数在每个批次中逐步更新，使得模型能够学习到更好的特征表示，从而提高在验证集或测试集上的性能。训练过程中的损失值变化能够帮助我们判断模型是否在有效地学习，以及是否存在过拟合或欠拟合的情况。

def train_model(model, train_dataloader, optimizer, loss_fn, num_epochs):
    model.train()
    for epoch in range(num_epochs):
        for inputs, labels in train_dataloader:
            inputs = inputs.to(device)
            labels = labels.to(device)
            logits = model(inputs)
            loss = loss_fn(logits, labels)

            optimizer.zero_grad()
            loss.backward()
            optimizer.step()

        print(f'Epoch [{epoch+1}/{num_epochs}], Loss: {loss.item():.4f}')

import torch
import numpy as np
from sklearn.metrics import accuracy_score, f1_score, roc_auc_score
from PIL import Image
import matplotlib.pyplot as plt

def evaluate_ensemble(models, test_dataloader, device):
    true_labels = []
    pred_probs = []

    with torch.no_grad():
        for inputs, labels in test_dataloader:
            inputs = inputs.to(device)
            labels = labels.to(device)
            ensemble_logits = torch.zeros((inputs.size(0), 4)).to(device)
            for model in models:
                model.eval()  # Ensure model is in evaluation mode
                logits = model(inputs)
                ensemble_logits += logits
            ensemble_probs = torch.softmax(ensemble_logits / len(models), dim=1)

            true_labels.extend(labels.tolist())
            pred_probs.extend(ensemble_probs.tolist())

    true_labels = np.array(true_labels)
    pred_probs = np.array(pred_probs)

    acc = accuracy_score(true_labels, np.argmax(pred_probs, axis=1))
    f1 = f1_score(true_labels, np.argmax(pred_probs, axis=1), average='macro')
    auc = roc_auc_score(true_labels, pred_probs, multi_class='ovo')

import torch
import numpy as np
from sklearn.metrics import accuracy_score, f1_score, roc_auc_score
from collections import Counter

def evaluate_ensemble_voting(models, test_dataloader, device):
    true_labels = []
    pred_labels = []

    with torch.no_grad():
        for inputs, labels in test_dataloader:
            inputs = inputs.to(device)
            labels = labels.to(device)
            batch_pred_labels = []
            
            for model in models:
                model.eval()  # Ensure model is in evaluation mode
                logits = model(inputs)
                probs = torch.softmax(logits, dim=1)
                batch_pred_labels.append(torch.argmax(probs, dim=1).cpu().numpy())
            
            # Transpose to get predictions of each sample across all models
            batch_pred_labels = np.array(batch_pred_labels).T
            
            # Voting: Choose the most common prediction for each sample
            voted_labels = [Counter(pred).most_common(1)[0][0] for pred in batch_pred_labels]
            
            true_labels.extend(labels.cpu().tolist())
            pred_labels.extend(voted_labels)

    true_labels = np.array(true_labels)
    pred_labels = np.array(pred_labels)

    acc = accuracy_score(true_labels, pred_labels)
    f1 = f1_score(true_labels, pred_labels, average='macro')
    auc = roc_auc_score(true_labels, np.eye(4)[pred_labels], multi_class='ovo')

    print(f'Test accuracy: {acc:.4f}')
    print(f'Test F1 score: {f1:.4f}')
    print(f'Test AUC score: {auc:.4f}')

# 假设 models 是之前训练好的多个模型的列表
# 假设 device 已经被定义
evaluate_ensemble_voting(models, test_dataloader, device)

from sklearn.metrics import accuracy_score, f1_score, roc_auc_score, confusion_matrix
def evaluate_ensemble(models, test_dataloader):
    true_labels = []
    pred_probs = []
    pred_labels = []

    with torch.no_grad():
        for inputs, labels in test_dataloader:
            inputs = inputs.to(device)
            labels = labels.to(device)
            ensemble_logits = torch.zeros((inputs.size(0), 4)).to(device)
            for model in models:
                logits = model(inputs)
                ensemble_logits += logits
            ensemble_probs = torch.softmax(ensemble_logits / len(models), dim=1)
            ensemble_preds = torch.argmax(ensemble_probs, dim=1)

            true_labels.extend(labels.tolist())
            pred_probs.extend(ensemble_probs.tolist())
            pred_labels.extend(ensemble_preds.tolist())

    true_labels = np.array(true_labels)
    pred_probs = np.array(pred_probs)
    pred_labels = np.array(pred_labels)

    acc = accuracy_score(true_labels, pred_labels)
    f1 = f1_score(true_labels, pred_labels, average='macro')
    auc = roc_auc_score(true_labels, pred_probs, multi_class='ovo')

    print(f'Test accuracy: {acc:.4f}')
    print(f'Test F1 score: {f1:.4f}')
    print(f'Test AUC score: {auc:.4f}')
